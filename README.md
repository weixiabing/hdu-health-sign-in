# hdu-health-sign-in
# 觉得好用的麻烦给个star
+ 杭电健康打卡脚本github action食用 简单方便

+ settings-secrets添加USERNAME和PASSWORD(即为易班app的登陆账号(账号为手机号)密码)
+ 支持多账号填入USERNAME和PASSWORD时每个号用&隔开(如username:12322&1221112&1124113)

+ 每天7点自动打卡(若杭电助手还是发来打卡通知说明打卡失效(若打卡失效请及时联系作者)或账号错误)
+ tg通知功能
+ 新增error记录功能
+ 本地部署版即将上线
